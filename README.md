Разработка выполнялась в урезанной версии 1с, поэтому работа с интрефейсом или регламентными заданиями была недоступна, по сути своей функционал представляет собой две хозяйственные операции: составление договора и его оплата. По ним строятся отчеты: не оплаченные договора, и сумма по агентам и по филлиалам. Есть возможность указывать сумму в валюте по курсу ЦБР.
